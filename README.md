# Basic Range React component

Basic Range component build with React

- The examples can be found in `src/examples/`;
- The component Range can be found in `src/components/Range`;

## Installation

```
npm install
```

## Run the examples on port 3000

```
npm run develop
```

## Test

```
npm run test
```

## Lint code

```
npm run lint
```

## Format code

```
npm run format
```
